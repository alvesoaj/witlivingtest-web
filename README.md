## Development Setup (Not needed to test APP)

### With NPM installed, Installing Bower:

`npm install -g bower`

### First setup Bower:

`bower init`

### Create .bowerrc with:

```
{
    "directory": "vendor/assets/components",
    "ignoredDependencies": [
        "jquery"
    ]
}
```

### Installing Bootstrap:

`bower install bootstrap --save`
